Even Odd

Que4: You are given an array A containing N integer elements, and your task is to return an array B ( having a size equal to 2),where B[0] contains the sum of all even elements of array A and B[1] has the sum of all odd elements of the array A.
Note: You have to complete findEvenOdd function. No need to take any input.

Ans: 
            let oddCount = 0;
            let evenCount = 0;
            let arrB = [];
            for(let i of Arr){
              if(i%2===0){
                evenCount+=i;
              }else{
                oddCount+=i;
              }
            }
            arrB[0] = evenCount;
            arrB[1] = oddCount;
            return arrB;
